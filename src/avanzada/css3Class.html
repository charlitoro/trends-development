Style Sheet (CSS) o Holas de Estilo en Cascada en su traducción al español, al igual que HTML es un lenguaje de marcado que nos permite aplicar estilos dentro dentro de nuestras etiquetas HTML.

Es común encontrar la analogía con vehículo con un diseño antiguo y uno con diseño moderno, donde el auto viejo es el HTML simple pero cuando se le aplica estilos CSS ya tenemos un auto que a simple vista pareciera nuevo.

Selectores

Un selector es la forma en como nosotros le decimos a las etiquetas que estilos deben tomar. Existen varios selectores:

  • Selector de tipo: Es el tipo más utilizado donde se refiere siempre a una determinada etiqueta HTML. 

                        h1 {
                        color: #000033;
                    }

                    /* Es posible utilizar más etiquetas */
                    h1, h2, h3 {
                        color: #000033;
                    }

  • Selector de clase: En ocasiones a un estilo definido para una etiqueta se le necesita añadir algunos estilos adicionales es una sola sección de tu página. Para darle solucion a esto estan las clases, donde solo se debe añadir el nombre de la clase después de el nombre de la etiqueta, separados por un punto: 

                        h2.rojo-negrita {
                        color: #330000;
                        font-weight: bold;
                    }

                    /* */
                    .rojo-negrita {
                        color: #330000;
                        font-weight: bold;
                    }

  • Selector de ID: Los selectores ID le permiten asignar nombres significativos a elementos HTML concretos. Los selectores ID se utilizan para un solo elemento: 

                        h1#verde-negrita {
                        color: #003300;
                    }

                    /* */
                    #verde-negrita {
                        color: #003300
                    }

  • Selector universal: Sirve para seleccionar, con ayuda del comodín *, cada elemento del árbol de documento. 

                        * {
                        margin: 0;
                        padding: 0;
                    }

Importación de estilos

Para poder importar y utilizar nuestros estilos sobre las etiquetas HTML se lo puede hacer principalmente de 3 maneras:

  • Directamente en la etiqueta: En cada etiqueta podemos utilizar la palabra style="" y dentro de las comillas pueden ir los estilos: 

                        <header style="background-color: darkviolet;">
                    </header>

  • Etiqueta <Style>: La etiqueta <style> que se comunmente de ubica dentro del <head> en la estructura HTML, puede guardar los estilos CSS: 

                        <style>
                        h1 {
                            font-family: 'Gill Sans', 'Gill Sans MT', Calibri, 'Trebuchet MS', sans-serif;
                            color: darkslategray;
                        }
                        strong {
                            color: teal;
                        }
                    </style>

  • Desde un archivo .css: Esta es la manera que se aconseja utilizar porque nos permite tener un proyecto organizado sin cargar mucho nuestras paginas HTML. Dentro de un archivo.css puedes guardar tus estilos e importalos con la etiqueta <link> 

                        <link rel="stylesheet" href="css/styles.css">

    El parámetro href es donde guardas la direccion donde se encuentra tu archivo .css.
